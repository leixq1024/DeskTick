# Changelog

All notable changes to DeskTick will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-28

### Added
- Initial release of DeskTick
- Real-time stock price monitoring
- Support for Chinese A-shares (Shanghai and Shenzhen)
- Support for US stocks
- Transparent, always-on-top desktop widget
- Draggable window with position saving
- Configuration system with JSON persistence
- Multi-stock monitoring capability
- Color-coded price changes (red/green)
- Auto-refresh at configurable intervals
- Cross-platform support (Windows, Linux, macOS)
- Example configuration file
- Comprehensive test suite
- Launcher scripts for all platforms
- Full documentation in README

### Technical Details
- Built with Python 3.7+ and PyQt5
- Uses Sina Finance API for stock data
- MIT License
- Proper packaging with setup.py and MANIFEST.in
